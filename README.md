# 🦜 ParrotOS Portfolio

A creative portfolio website styled as a ParrotOS Linux desktop environment. Built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- **Desktop Environment** — Full ParrotOS-style desktop with draggable windows
- **Interactive Terminal** — Working terminal with custom commands (`neofetch`, `help`, `skills`, etc.)
- **File Manager** — Folder navigation system for projects
- **Window Management** — Minimize, maximize, close, drag & resize windows
- **Start Menu** — Application launcher with search
- **Taskbar** — Active windows, system tray, clock & calendar
- **Context Menu** — Right-click desktop menu

## 📁 Project Structure

```
├── index.html          # Main HTML file
├── css/
│   ├── desktop.css     # Desktop & icons styles
│   ├── windows.css     # Window styles & file manager
│   ├── taskbar.css     # Taskbar & start menu
│   └── terminal.css    # Terminal styles
├── js/
│   ├── main.js         # App initialization
│   ├── desktop.js      # Desktop & folder navigation
│   ├── window-manager.js   # Window management
│   └── terminal.js     # Terminal emulator
└── README.md
```

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript** — Vanilla ES6+, no frameworks
- **Fonts** — JetBrains Mono, Ubuntu
